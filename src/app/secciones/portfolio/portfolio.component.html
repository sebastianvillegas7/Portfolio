MoodShare App

MoodShare es una aplicación inspirada en mi amor por la música y el deseo de compartirlo con los demás. Su objetivo principal es ofrecer una plataforma para compartir gustos musicales con otros usuarios, por ejemplo, entre los empleados de una compañía, alumnos de una institución, etc.
Desarrollada con Angular 16, Spring Boot y el servicio API de Discogs.
